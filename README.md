# Platinum-Lang
A programming language made with Rust

# How to run

## Tests

If you want to run tests use this code.

```bat
.\project test
```

## Run the CII

It returns only statements in string format.

```bat
.\project run
```

# For pull requesters

## Use `project.bat` for fast git push

```bat
.\project push <The name of pull request>
```
